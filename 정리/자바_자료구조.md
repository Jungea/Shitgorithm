# 자바_자료구조

## QUEUE
- Last(Rear) -> ㅁㅁㅁㅁ -> First(Front)
- offer, poll, peek

```java
import java.util.ArrayDeque;

Queue<Integer> queue = new ArrayDeque<>();

queue.offer();
queue.poll();
queue.peek();

```

## DEQUE
- 양방향 입출력
- Last(Rear) <-> ㅁㅁㅁㅁ <-> First(Front)
- offer, poll, peek

```java
import java.util.ArrayDeque;

Deque<Integer> deque = new ArrayDeque<>();

// Queue : Enqueue -> (Rear) ㅁㅁㅁㅁ (Front) -> Dequeue
deque.offerLast();
deque.pollFirst();
deque.peekFirst();


// Stack : push/pop <-> (Front) ㅁㅁㅁㅁ (Rear)
deque.offerFirst();  // deque.push():
deque.pollFirst();  // deque.pop():
deque.peekFirst();  // deque.peek():

```

## PriorityQueue
- 우선순위큐
- 정의한 우선순위에 따라 우선순위가 높은 값이 먼저 제거

```java
PriorityQueue<Integer> pq = new PriorityQueue<>(Comparator.reverseOrder());  // (a, b) -> b - a
pq.offer(x);
pq.poll();
```
