# 자바_자료구조

## DEQUE
- First ㅁㅁㅁㅁ Last
- offer, poll, peek

```java
import java.util.ArrayDeque;

Deque<Integer> deque = new ArrayDeque<>();

// Queue : Enqueue - (Rear) ㅁㅁㅁㅁ - (Front) - Dequeue
deque.offerLast();
deque.pollFirst();
deque.peekFirst();

```

## Map
- <Key, Value>

```java
Map<String, Integer> map = new HashMap<>();

map.put("aaa", 1);
map.get("aaa");
map.containsKey("aaa");

map.getOrDefault("aaa", 0);
List<Integer> values = map.values();
Set<String> keys = map.keySet();

for (Map.Entry<String, Integer> e : map.entrySet()) {
```
