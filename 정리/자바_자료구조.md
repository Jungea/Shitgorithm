# 자바_자료구조

## DEQUE
- First ㅁㅁㅁㅁ Last
- offer, poll, peek

```java
import java.util.ArrayDeque;

Deque<Integer> deque = new ArrayDeque<>();

// Queue : Enqueue - (Rear) ㅁㅁㅁㅁ - (Front) - Dequeue
deque.offerLast();
deque.pollFirst();
deque.peekFirst();

```

## PriorityQueue
- 우선순위큐
- 정의한 우선순위에 따라 우선순위가 높은 값이 먼저 제거

```java
PriorityQueue<Integer> pq = new PriorityQueue<>(Comparator.reverseOrder());  // (a, b) -> b - a
pq.offer(x);
pq.poll();
```
