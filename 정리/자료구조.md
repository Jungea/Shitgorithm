# 3주차 개념 총정리
스택(Stack) / 큐(Queue) / 힙(Heap) / 해시(Hash)

---
## 1. 스택 (Stack)

### 핵심 개념
- LIFO (Last In First Out)
- 최근에 넣은 값이 먼저 나온다.

### Java 구현
```java
// ❌ Stack 클래스는 레거시
Deque<Integer> stack = new ArrayDeque<>();
stack.push(10);      // 삽입
stack.pop();         // 제거 & 반환
stack.peek();        // 조회 (제거 X)
stack.isEmpty();
```

### 사용되는 상황
- 뒤로가기(Undo)
- 괄호 검사
- DFS(비재귀)
- 수식 계산
- 최근 요소가 중요할 때

### 시간복잡도
- push: O(1)
- pop: O(1)
- peek: O(1)

### 코테 대표 패턴
- 괄호 매칭
- 단조 스택(Monotonic Stack)
- 스택 시뮬레이션 (예: 1874 스택 수열)

---

## 2. 큐 (Queue)

### 핵심 개념
- FIFO (First In First Out)
- 먼저 들어온 데이터가 먼저 나온다.

### Java 구현
```java
Queue<Integer> queue = new LinkedList<>();
// 또는
Deque<Integer> queue = new ArrayDeque<>();  // 더 빠름

queue.offer(10);     // 삽입
queue.poll();        // 제거 & 반환
queue.peek();        // 조회
```

### 사용되는 상황
- BFS 알고리즘
- 프로세스 / 대기열 시뮬레이션
- 순차적인 작업 처리

### 시간복잡도
- enqueue: O(1)
- dequeue: O(1)

### 코테 대표 패턴
- BFS 미로 탐색
- 프린터 큐 문제
- 원형 큐 / 데크(deque) 활용

---

## 3. 힙 (Heap) — 우선순위 큐

### 핵심 개념
- 최대/최솟값을 빠르게 꺼내는 자료구조
- 완전 이진 트리 구조
- 최소 힙 / 최대 힙 존재

### Java 구현
```java
// 최소 힙 (기본)
PriorityQueue<Integer> minHeap = new PriorityQueue<>();

// 최대 힙
PriorityQueue<Integer> maxHeap = new PriorityQueue<>(Collections.reverseOrder());

minHeap.offer(10);   // 삽입
minHeap.poll();      // 최솟값 제거 & 반환
minHeap.peek();      // 최솟값 조회
```

### 사용되는 상황
- Top K 문제
- 가장 큰/작은 값 반복 추출
- 다익스트라 알고리즘
- 빈도 기반 정렬 문제

### 시간복잡도
- push: O(log N)
- pop: O(log N)
- peek: O(1)

### 코테 대표 패턴
- K번째 큰 수
- Top K Frequent Elements
- 우선순위 기반 시뮬레이션

---

## 4. 해시 (Hash) — HashMap / HashSet

### 핵심 개념
- key-value 구조로 구성된 자료구조
- 평균적으로 O(1) 삽입/탐색

### Java 구현
```java
// 빈도 계산
Map<String, Integer> freq = new HashMap<>();
freq.put("key", freq.getOrDefault("key", 0) + 1);

// 중복 제거, 존재 확인
Set<Integer> set = new HashSet<>();
set.add(10);
set.contains(10);  // true
```

### 사용되는 상황
- 중복 체크
- 등장 횟수 카운팅
- 특정 값 존재 여부 확인
- 애너그램 판별

### 시간복잡도
- put: O(1)
- get: O(1)
- remove: O(1)

### 코테 대표 패턴
- 완주하지 못한 선수
- 빈도 계산 문제
- 중복 없는 원소 관리
- 해시 + 투포인터/슬라이딩 윈도우 조합 문제

## 3주차 핵심 요약

| 자료구조 | 핵심 성질 | 대표 문제 패턴 |
|---------|-----------|----------------|
| 스택 | LIFO | 괄호 검사, 단조 스택, 스택 시뮬레이션 |
| 큐 | FIFO | BFS, 프린터, 대기열 시뮬레이션 |
| 힙 | 우선순위 | Top K, 빈도 정렬, 최소/최대값 반복 처리 |
| 해시 | 빠른 탐색 | 빈도 계산, 중복 체크, 애너그램 |
